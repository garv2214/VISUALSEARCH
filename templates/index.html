Visual Search OCR

A powerful command-line tool that extracts text from images using advanced OCR and visual search capabilities.

Get Started View on GitHub
VisualSearch.py 

# Capture and search an image
python VisualSearch.py

# Perform OCR and search
python VisualSearch.py --ocr

# Analyze an image
python VisualSearch.py --analyze

Key Features

Live Camera Capture

Capture and process images directly from your webcam with a simple command-line interface.

Advanced OCR

Extract accurate text from images using the powerful Tesseract OCR engine with preprocessing for better results.

Visual Search

Automate Google Lens searches directly from the terminal for visual queries and object identification.

Image Analysis

Analyze image quality, blur, brightness, dominant colors, and edge density for optimal results.

Command-Line Interface

A simple and intuitive command-line interface for quick and efficient visual searches.

Debug Information

Access detailed debugging information about your system and dependencies to troubleshoot issues.

Installation

1

Clone the Repository

git clone https://github.com/garv2214/VISUALSEARCH.git
cd VISUALSEARCH
2

Requirements

pip install -r requirements.txt

Requires Python 3.6+

3

Run the Application

python VisualSearch.py
